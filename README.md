# 소개

페이스북과 네이버 밴드를 모티브로 만든 그룹형태의 SNS 서비스. Nest.JS 와 React.JS 를 활용하여 구현하였다.

 - Git 버전관리

# 환경설정 
<pre>
> git clone https://github.com/TeamArchive/Archive_SNS.git

Backend
> cd server & npm install
> npm run start

Frontend
> cd ../app & npm install
> npm start
</pre>
